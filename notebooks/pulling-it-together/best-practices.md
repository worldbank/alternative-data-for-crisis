# File Management and Code Dissemination
## Summary
In this section, we present the best practices for file management and code dissemination that we follow at the World 
Bank Data Lab.

After having participated in several Data Science projects, the Data Lab team created 
[this GitHub repository template](https://worldbank.github.io/template/README.html) to be used as a starting point for 
each of the projects we work on. Moreover, the repository serves as a guideline of best practices for managing Data 
Science projects. 

## Learning Objectives
### Overall goals
The main goal of this class is to teach students how to manage data files, document code, and disseminate work as a web book.  

### Specific goals
At the end of this notebook, you should have gained an understanding and appreciation of the following:
1. **GitHub Template**:
   - How to create a GitHub repository.
   - Understand GitHub workflow and best practices.
   - Create your project GitHub repository.
2. **Data Management**:
   - Understand how to manage and store data in a secure way. 
   - Understand how to handle data versioning and its challenges.
